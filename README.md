# Pixabay Clone 📸

A simple and responsive Pixabay Clone built using React and Vite.  
Users can search and explore beautiful images using the Pixabay API.

---

## 🚀 Features

- 🔎 Search images by keyword
- 🎲 Category based image filtering
- 📱 Responsive UI
- ⚡ Fast performance using Vite
- 🌐 Fetching real-time images from Pixabay API
- 🧠 React Context API for state management
- 🎨 Clean and modern interface

---

## 🛠️ Technologies Used

- React.js
- Vite
- Bootstrap
- Pixabay API
- Context API
